4

Rule: Output = Input + 4

Rule: Output = Input x 4
Addition Multiplication
Answers and working out
How to complete the tablesPick any number (e.g. 4). For addition, add that number to every input: 0+4=4, 1+4=5, 2+4=6, etc. For multiplication, multiply every input by that number: 0x4=0, 1x4=4, 2x4=8, etc.
Describe the addition patternThe output goes up by the same amount every time. The jump between outputs is always equal to your chosen number. This is a constant increase.
Describe the multiplication patternThe output goes up by more each time. The gap between outputs keeps growing. This is because you are multiplying by increasingly larger inputs.

Compare
4
Addition (+ 4)
The jump between each output is always the same.
Multiplication (x 4)
The jump between each output gets bigger every time.
How are they similar?
Both patterns increase as the input goes up. Both use the same chosen number. Both produce a predictable pattern you can continue.
How are they different?
Answers and working out
a) How are they similar?Both patterns increase as the input goes up. Both use the same number you chose. Both start from a predictable point and follow a rule you can continue.
b) How are they different?Addition increases by the same amount every time (constant jumps). The bars are all the same height. Multiplication increases by more and more each time (growing jumps). The bars get taller and taller. That is why multiplication grows much faster than addition.
